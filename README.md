# Exam

## 題目:
 
輸入一個英文字, 顯示中文意義.

## 功能:
1. 新增資料(中文和英文)	
2. 修改資料(修改新增後的資料)
3. 刪除資料(刪除目前的資料)
4. 顯示資料(顯示新增後的資料,以 Table 方式顯示, 如果有分頁更好)
5. 搜尋資料(輸入中文或英文關鍵字,  顯示該關鍵字在資料庫中的資料, 同顯示資料方式顯示).
